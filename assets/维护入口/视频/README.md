# 视频维护

把 `.mp4`、`.webm`、`.ogg` 或 `.mov` 视频文件直接放进这个文件夹，运行根目录的 `刷新维护入口.ps1` 后会自动显示在网页的视频模块里。

- 文件名会作为切换按钮标题，开头序号只用于排序。
- 可放置同名 `.txt` 覆盖文字，例如 `01-demo.mp4` 搭配 `01-demo.txt`。
- 同名图片可作为封面，例如 `01-demo.jpg`。
- `.txt` 可用字段：`title`、`tag`、`text`。
- 示例 `01-rainbow-demo.txt` 使用 `type=gradient` 生成 5 秒彩虹渐变演示。
